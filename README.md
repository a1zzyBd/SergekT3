# Проекты: Парсер парковок Алматы и WhatsApp-рассыльщик

## Описание

Данный репозиторий содержит два учебных проекта:

1. **Парсер парковок Алматы** — собирает информацию о парковках из API 2GIS и сохраняет её в CSV-файл.
2. **WhatsApp-рассыльщик** — отправляет сообщения из CSV-файла через WhatsApp с поддержкой безопасного режима проверки.

---

# 1. Парсер парковок Алматы

## Возможности

* Поиск парковок через API 2GIS;
* Использование нескольких поисковых запросов;
* Сбор информации:

  * название;
  * адрес;
  * координаты;
  * ссылка на 2GIS;
  * режим работы;
* удаление дубликатов;
* сохранение результата в CSV.

## Быстрый запуск (5 минут)

### Установка зависимостей

```bash
pip install pandas requests
```

### Настройка

В файле `parking_parser.py` укажите свой API-ключ 2GIS:

```python
API_KEY = "ВАШ_API_КЛЮЧ"
```

Получить ключ можно после регистрации на платформе разработчиков 2GIS.

### Запуск

```bash
python parking_parser.py
```

### Результат

После выполнения будет создан файл:

```
almaty_parking.csv
```

# 2. WhatsApp-рассыльщик

## Возможности

* Чтение номеров и сообщений из CSV;
* Проверка корректности номеров Казахстана;
* Поддержка двух режимов работы:

  * `dry-run` — только проверка без отправки;
  * `live` — реальная отправка сообщений;
* Ограничение количества отправленных сообщений;
* Ведение журнала в тестовом режиме.

## Важно

Проект создан исключительно в демонстрационных и учебных целях.

Массовая рассылка сообщений незнакомым пользователям запрещена правилами WhatsApp и может привести к блокировке аккаунта.

---

## Быстрый запуск (5 минут)

### Установка зависимостей

```bash
pip install pandas pywhatkit
```

### Подготовка CSV-файла

Создайте CSV-файл следующего формата:

| phone       | message            |
| ----------- | ------------------ |
| 87071234567 | Привет!            |
| 87771234567 | Тестовое сообщение |

Пример:

```csv
phone,message
87071234567,Привет!
87771234567,Тестовое сообщение
```

Укажите путь к файлу в коде:

```python
df = pd.read_csv("message.csv", encoding="utf-8-sig")
```

---

## Режим проверки (рекомендуется)

Перед реальной отправкой включите безопасный режим:

```python
MODE = "dry-run"
```

Запуск:

```bash
python whatsapp_sender.py
```

Программа покажет, какие сообщения были бы отправлены, и сохранит информацию в файл:

```
log.txt
```

---

## Режим реальной отправки

Для отправки сообщений:

```python
MODE = "live"
```

При необходимости можно ограничить количество отправок:

```python
MAX_LIVE_MESSAGES = 2
```

Перед запуском убедитесь, что:

* установлен WhatsApp Web;
* выполнен вход в аккаунт;
* браузер открыт и имеет доступ к интернету.

Запуск:

```bash
python whatsapp_sender.py
```
